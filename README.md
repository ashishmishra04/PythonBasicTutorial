# Variables
| Type                 | Description                                      | Use Case                              |
| -------------------- | ------------------------------------------------ | ------------------------------------- |
| **bytearray**        | Mutable sequence of bytes                        | Network or file I/O buffers           |
| **memoryview**       | Efficient view of binary data                    | Avoids data copying for large buffers |
| **complex**          | Represents numbers with real and imaginary parts | Scientific/math computations          |
| **generator**        | Lazy iterable created with `()` or `yield`       | Large or infinite sequences           |
| **slice**            | Defines range for slicing (`[start:stop:step]`)  | Used internally in sequence slicing   |
| **property**         | Encapsulates attribute access                    | OOP getter/setter management          |
| **NotImplemented**   | Tells Python an operation isnâ€™t supported        | Used in operator overloading          |
| **Ellipsis (`...`)** | Placeholder or advanced slicing                  | NumPy or deep slicing syntax          |
| **compile()**        | Compiles code strings                            | Dynamic code execution or caching     |
| **frozenset**        | Immutable version of set                         | Dictionary keys or constant sets      |
